# QuickTask 📝
A lightweight task management web app for quickly capturing tasks and organizing them simply. The goal is to remove friction from remembering things.

Status: Early development

## Features
- Quick task creation
- Task organization with folders
- Fast and minimal interface
- Offline storage

## Tech Stack
- React
- React Query
- IndexedDB (Dexie)
- TailwindCSS

## Run Locally
1. Clone the repo
```
git clone https://github.com/alborzasd/quick-task.git
```
2. Change directory to the repo
```
cd quick-task
```
3. Install packages
```
npm install
```
4. Run dev server
```
npm run dev
```
5. You can open the browser and enter the address `http://localhost:5173`. The **port number** will be specified when the dev server starts.
